**EyrieHQ** is a self-hosted infrastructure intelligence platform for AWS — built for DevOps engineers, SREs, and platform teams who want real visibility and **zero standing access**.

- **Real-time AWS monitoring** — EC2, EKS, RDS, ElastiCache, DocumentDB, OpenSearch, SES, Secrets Manager, and more
- **Full-stack observability** — logs, metrics, and distributed traces via an OpenTelemetry-native agent
- **Just-in-time access control** — request-approval workflows with auto-expiring AWS STS credentials
- **Plugin architecture** — 15 built-in plugins, open SDK, extensible by design

No vendor lock-in. No per-host pricing. AGPLv3 — self-host it, own your data.
